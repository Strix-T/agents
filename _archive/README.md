# Archive

Retired agents live here. Don't delete them — they're useful reference when designing new agents or remembering why a previous approach didn't work.

When archiving an agent:
1. Move the file from the repo root into this folder.
2. Add a one-line note at the top explaining why it was retired and when.
3. Remove the symlinks from `~/.claude/agents/` and any project's `.cursor/rules/`.
4. Remove the entry from the Roster table in the main README.
